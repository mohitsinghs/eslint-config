## Usage

Install with your preferred package manager

```sh
# with npm
npm i -D eslint prettier @mohitsingh/eslint-config
# or with yarn
yarn add --dev eslint prettier @mohitsingh/eslint-config
```

Add the following eslint config in your package.json

```json
"eslintConfig": {
  "extends": "@mohitsingh"
}
```
